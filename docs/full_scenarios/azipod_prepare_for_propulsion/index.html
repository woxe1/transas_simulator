Model C (AZIPOD) Preparing for propultion

  • 4D → Engine 1,2,3 LOP → Убеждаемся, что все ручки Remote

  • AMS → GENERATORS → POWER OVERVIEW → PMS in Manual → GENERATORS → Start DG2

  • 4B → Medium Voltage MCD → Close 121, 123, 222, 254, 252, 151, 153

  • 4D → AFT SW Pumps → Run, remote (all)

  • 4D → AFT Central FW Cool.Pumps → Run, remote (all)

  • 3D → FWD SW Pumps → → Run, remote (all)

  • 6A → Azipod Room Ventilation → Run, remote (all)

  • 6A → Cycloconverter PORT (&STBD) → TAKE OVER → Start/stop

  • 6A → Azipod Steering PORT (&STBD) → All remote and auto

  • 6A → Azipod Aux Starter PORT (&STBD) → All in auto

  • 6A → Cycloconverter PORT (&STBD) → Start all AQ&BQ → TAKE OVER

  • ECR → Azipod Control → Azipod Port(&STBD) Responsibility to ECR → ECR IN CONTROL (stop blinking) →

  • ACS → Control Panel → Drive start (all)

  • ECR → Azipod Control → Даём малый ход

  • 3D → DG LT CWS Pump → Start all & remote

  • 4D → DG LT CWS Pump → Start all & remote

An alternative way to start a diesel generator:

  • 4D → Engine 1,2,3 LOP → DG2 Local → Start → DG2 Remote

  • 4B → AFT DG 1,2,3 MSWB → GEN2 Manual

  • 4B → AFT MSWB 1& Synchro → All Local → Synchronizing & Generator control ON, Gen2 → Raise → Close

One more alternative way to start a diesel generator:

  • 4B → AFT DG 1,2,3 MSWB → GEN2 Manual → Start → ON/OFF → Remote
