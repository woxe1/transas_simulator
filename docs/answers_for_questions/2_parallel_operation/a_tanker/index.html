Model A TANKER

Start

For starting drive engine a few precautions should be taken:

  • We should check engine parameters. That's parameters depend on the current condition of vessel for example if some maitenance was taken on generator before working with it, we should check that the air is opened.
  • Also we should check thet the fuel valves is opened and the fuel is available.
  • We should check that the cooling water is available if the engine has water cooling
  • Also recomended to start priming pump if it doesn't work automatically
  • You should check that there is no alarms on Local control panels and reset them if they are. In this model it done by help of lever on the LOP. ER3 â†’ GE LOP

After start we should wait for the engine to reach the required speed about eight hundred RPM.

Stopping

Engine doesn't stop in moment. Drive engine will work for cooling it's parts and then it will stop.

Gen connection to the dead bus

Connection to the dead bus we can do without control only one parameter - that the generator was connected correctly in according with scheme for providing the correct phase sequence.

Gen connection for parallel operation

When we are going to connect generator to work in parallel operation we should check four parameters: - phase shift of connecting generator (by means of synchronoscope) - voltage of connecting generator (by means of voltmeter) - frequency of connecting generator (by means of frequency meter) - phase sequence (by means of phase sequence indicator it's absent on this model but we can check connections)

Disconnection

For disconnection your generator should have less then 10% of it's nominal load for prevent a big currents on the working generator. After that we can disconnect it from the bus.

load transfer/sharing

We can share load between generators by means of automatic load-sharing system. That system change the value of fuel feed to the engine so can change the active power of the generator. Also we can do it by means manual control with help of load-sharing levers. When we transfer load from one generator to another we should control frequncy of the electrical system. And when we raise fuel feed with help of one governor handle we sholud decrease the fuel feed with the other governor handle.
